guide · 使用指南

这是什么

把移动端 UI 截图、风格参考图、一段功能描述喂给 Claude Code，自动产出可交互的单页 HTML demo。Skill 用 Gemini 生成贴合风格的素材，用结构化 schema 推导布局。

安装

  1. 下载上方 mobile-ui-replicator.zip 解压
  2. 把整个文件夹放在你想用它的项目根目录下
  3. 打开里面的 .env，把 GEMINI_API_KEY 占位值换成你的真 key（aistudio.google.com/apikey 免费申请）
  4. 装 Python 依赖：pip install pillow scipy numpy（可以让 AI 执行）

依赖

  • 必须：Python 3.10+、Gemini API key
  • 软依赖（强烈推荐）：Claude Code 的 frontend-design skill。写 HTML 那步会自动调它做 polish — 动效、间距、微交互的精致度差一档。没装也能跑，只是结果不那么"shippable"

使用

在 Claude Code 对话里，把界面截图、风格参考图、功能描述三个东西连提示词一起发给 AI：

读 ./mobile-ui-replicator/SKILL.md
帮我把 ./my-ui.png + ./style.jpg 做成 demo，名字叫 cafe-finder

功能描述：
- 这是个咖啡馆发现 app，主屏首页
- hero 是大图 + 标题"今日推荐"
- 下面是分类瓷砖（手冲/拿铁/甜品）
- 再下面是横向滚动的"附近的店"卡片
- 底部 tab 栏：发现 / 收藏 / 我的

Agent 跑完产出在 ./mobile-ui-replicator/output/cafe-finder/index.html，典型耗时 1–3 分钟。

输出说明

  • 单 HTML 文件，CSS / JS 全内联，双击即开
  • iPhone 竖屏壳固定 390 × 844，含 notch、纸纹、edge vignette
  • 5 层 z-axis 结构：topbar / background or content / secondlayer / floating / sheet or bottombar，分层定位
  • 素材都是 Gemini 即时生成的 PNG；mascot 类会自动 flood-fill 抠透明
  • 字体走 Google Fonts CDN，中文优先 Noto Sans SC / LXGW WenKai Screen，agent 自动匹配参考
  • 交互用原生 HTML5 API：拖拽 / spring 滚动 / tab 切换 / localStorage 持久

已知限制

  • 仅支持 iPhone 竖屏 390 × 844，没做 Android / 平板 / 横屏
  • 仅做单屏 app，不做多屏跳转 / 登录流 / 嵌套子页
  • 素材数量上限约 15 张，超出会被 Gemini 限流
  • 复杂粒子或物理动画无法精准还原，用 CSS 关键帧近似；<video> 仅生成占位
  • 自定义品牌字体替换为最接近的 Google Font，不二次创作字体
  • 地图 / 海报类素材的图内文字一律走 CSS 叠加（Gemini 中文易乱码）
