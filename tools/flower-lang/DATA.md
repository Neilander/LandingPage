# 花数据（阶段 A 产出）

12-16 朵花，每条包含:
- `id` slug （英文无空格，画 SVG 时的 symbol id）
- `zh` 中文名 + `en` 英文名 + `latin` 学名
- `tags` 关键词数组（中英混合，搜索匹配用）
- `meaning` 花语主旨（一句）
- `quote` 一句诗意小注
- `palette` 用哪几个色
- `notes` 画法提示

## 轮次 1 产出（6 朵，先行）

### 1. rose-red · 红玫瑰
- **zh**: 红玫瑰 · **en**: Red Rose · **latin**: Rosa
- **tags**: 爱情 love 热情 passion 心动 romance 表白 confession 炽热
- **meaning**: 炽热的爱
- **quote**: 说不出口的都交给一朵红玫瑰。
- **palette**: brick（主）+ moss（茎叶）
- **notes**: 侧视一朵半开的玫瑰头 + 茎上两片叶子；花心螺旋重叠的几片花瓣；线稿为主，花瓣内侧用 brick 部分填充

### 2. violet-white · 白紫罗兰
- **zh**: 白紫罗兰 · **en**: White Violet · **latin**: Viola odorata
- **tags**: 谦逊 modesty 纯真 innocence 忠诚 faithfulness 秘密 secret 请相信我 trust
- **meaning**: 请相信我
- **quote**: 细小而坚定，像一个不肯说出口的承诺。
- **palette**: lavender（瓣尖）+ brass（花心）+ moss（叶）
- **notes**: 5 瓣紫罗兰（上 2 + 侧 2 + 下 1），下瓣带 3 条细紫脉

### 3. lily-of-valley · 铃兰

- **meaning**: 幸福终将归来
- **quote**: 低头才能看见的甜。

- **notes**: 弧形弯茎，挂 5-7 个小铃铛形花，连续线描

### 4. sunflower · 向日葵
- **zh**: 向日葵 · **en**: Sunflower · **latin**: Helianthus annuus

- **meaning**: 永远追光
- **quote**: 一直朝向太阳，是一种选择。

- **notes**: 圆盘花心外围 12-16 片椭圆花瓣；线稿轮廓 + 花盘纯色填充

### 5. forget-me-not · 勿忘我
- **zh**: 勿忘我 · **en**: Forget-Me-Not · **latin**: Myosotis

- **quote**: 蓝得那么小，却填满整个季节的空位。
- **palette**: lavender 或 sky（瓣）+ brass（花心）+ moss（叶）

### 6. peony · 牡丹
- **zh**: 牡丹 · **en**: Peony · **latin**: Paeonia
- **tags**: 富贵 prosperity 羞涩 bashfulness 幸福的婚姻 happy marriage 繁盛 flourish 丰盈
- **meaning**: 繁盛而羞涩的幸福
- **quote**: 所有重叠的花瓣里，藏着她不敢说的喜欢。

## 锁定色盘（CSS 变量 · 插图+工具通用）

```css
:root{
  --paper:     #f8f4ed;  /* 纸白底 */
  --ink:       #2a2520;  /* 深墨线稿 */
  --ink-dim:   #7a6f5f;  /* 次要文字 */
  --brick:     #c64a2a;  /* 砖红  */
  --moss:      #4a6b3a;  /* 苔绿（改深了一档,更有墨感） */
  --brass:     #d4a74a;  /* 黄铜 */
  --plum:      #6a3a5d;  /* 深紫 */
  --lavender:  #a38bc4;  /* 薰衣草 */
  --peach:     #e89a7a;  /* 奶杏 */
}
```

阶段 B 起, 所有 SVG 严格用这 6 个填充色 + 深墨线稿. 不加新色.
